## ハイライト

yakudoc の最初の公開リリースです。**依存ライブラリの JSDoc(型定義に書かれた英語ドキュメント)を、コードを 1 行も書き換えずにエディタ上だけ母語で表示**します。

```bash
npx yakudoc add zod
```

これだけで zod の API のホバーが日本語になります。公開済みのコミュニティ翻訳パックがあれば訳文まで自動で入ります。

## 含まれるもの

### CLI・コアライブラリ(`yakudoc`)

- `add` / `remove` — 依存ライブラリの型定義から JSDoc を抽出し、`.yakudoc/packs/` に翻訳パックを作成。コミュニティ翻訳パックがあれば訳文を自動適用
- `init` — tsconfig.json へのプラグイン登録 + 自分のコードの初回抽出
- `extract` / `status` / `translate` / `export` / `doctor`
- 翻訳先は日本語ほか 21 言語に対応
- 翻訳の照合は原文テキストのハッシュで行うため、ライブラリのバージョンが上がっても原文が変わっていない訳は生き続けます

### tsserver プラグイン(`yakudoc-ts-plugin`)

- ホバー・補完・シグネチャヘルプの documentation を翻訳済みテキストに差し替え
- `.yakudoc/translations.json` と `.yakudoc/packs/*.json` を監視し、変更を自動反映(エディタ再起動不要)

### 翻訳エンジン(いずれか一方)

- `yakudoc-mt` — オフラインの内蔵モデル(NLLB-200 / mBART-50)。API キー不要
- `yakudoc-ai-prep` — 任意の AI に翻訳させる下準備ファイルを生成

## 破壊的変更

なし(初回リリース)

## インストール

```bash
npm install --save-dev yakudoc yakudoc-ts-plugin
npx yakudoc init
npx yakudoc add <パッケージ名>
```
